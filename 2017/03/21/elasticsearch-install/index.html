虚拟机环境准备

  • 我是在wiodow下测试，所以使用vmware12，网络方式设置nat8模式，也可使用自带的hyper-v具体下载安装我这里不再叙述，需要详细安装步骤，请移步百度或谷歌。

  • 配置静态ip，在安装centos默认是动态ip获取，为了防止开机ip变化，可以把动态获取地址设置为静态，如果不需要设置静态ip跳过这步。

    • 使用nmcli命令 修改命令时 nmcli con mod 连接名 属性 值

  • jdk安装 网上很多教程，我这里下载解压在/usr/local/jdk1.8.0_121

    • 配置环境变量

    • 下载kibana地址地址https://www.elastic.co/downloads/kibana,请下载rpm包，我这里下载kibana-5.2.2-x86_64.rpm
